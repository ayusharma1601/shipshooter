Ship Shooter
A classic, fast-paced arcade shooter built with the Godot Engine. Take control of your ship, dodge obstacles, and blast through enemies to achieve the high score.

🕹️ How to Play Online
You can play the game directly in your browser without downloading anything:

👉 Play Ship Shooter Online

🎮 Controls
Left Arrow: Move ship to the left

Right Arrow: Move ship to the right

Spacebar: Fire weapons

🚀 Features
Instant Play: Fully exported for the web using WebAssembly.

Smooth Movement: Responsive ship handling for precise dodging.

Lightweight: Minimal assets for fast loading and performance.

🛠️ Built With
Godot Engine - The game engine used.

GDScript - For game logic and entity behavior.

WebAssembly/JS - For the web-based deployment version.
